## Enunciado

Leia um número inteiro e exiba a mensagem "Você digitou um número negativo." caso o número seja menor que zero.

Ex.:
```
Digite um número: -12
Você digitou um número negativo.
```
ou
```
Digite um número: 12
```
_(nada a exibir)_
